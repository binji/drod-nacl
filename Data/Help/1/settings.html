Changing Settings

The Settings Screen can be reached from the Main Menu by choosing the "Change Settings" option.  The following settings are available:

Personal - Enter your player name and optional CaravelNet information here.

Graphics - Click "Use full screen" to toggle between full screen and windowed mode (1024x768).  Note that you can also use the F10 key to do this at any time.  Turning off "Alpha blending" can reduce slowdown for certain in-game effects if you're experiencing a low framerate. However, it will also disable all those nice fade effects.

Sound - You may turn sound effects, game speech and background music on or off and set the volume each is played at. Unclick "Show subtitles when voices play" if you do not want subtitle text to be displayed in the game while characters are talking. If sound effects or music doesn't work, first try closing other applications and restarting DROD. If you are still without sound, try rebooting your computer.

Special - Select whether to play with room checkpoints or not, and whether to save demos at special points in the game. Checkpoints allow saving at certain locations in the middle of some rooms.

Keyboard commands - Use to redefine the game commands to different keys.  If you are using a non-US keyboard or have selected a layout other than Qwerty (i.e. Dvorak), you may see incorrect key descriptions as you redefine keys.  Despite this, the key redefinitions will probably work once you begin playing the game.  (Sorry.  We would like to fix this problem in a future release.)

Repeat Rate - Controls keyboard repeat rate (i.e. movement rate) during play.  Increasing it will decrease the delay between moves when a key is held down. Decreasing it will make movement animations smoother.

Editor - When "Save automatically" is checked, changes you make to a room in the editor are saved on leaving the room.  Turn this option off if you want to be prompted before saving any room changes.  Check "Show item tips" to receive help placing special objects in the room.
