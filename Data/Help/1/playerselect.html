Player Selection Screen

Choose "Change Player" on the Main Menu to access the Player Selection Screen. This screen has the following options available:

Okay - selects the highlighted player profile. Each player profile contains user preference settings and its own game progress.

New Player - creates a new player profile. This option is useful when you have more than one person playing DROD on this installation.

Delete player - permanently deletes the selected player profile. This includes deleting the player's saved games and demos (so make sure this is really what you want to do!).

Export - saves the active player profile to disk with all recorded demos and saved games. This option facilitates transferring your player settings and game information to a different DROD installation.

Export Saves Only - places all the current player profile's saved games in a file on disk with your player-specific information excluded. These saved games may then be imported by any player. Hint: use this option to share your progress with other DROD players without giving them author access to your levels.

Import - loads a DROD player profile exported to disk either with "Export" or "Export Saves only". This allows you to recover exported player saved games. Hint: to import saved games for a specific hold (set of levels), first import the hold into your installation of DROD, then import the player data.

Cancel - returns to the previous screen.
